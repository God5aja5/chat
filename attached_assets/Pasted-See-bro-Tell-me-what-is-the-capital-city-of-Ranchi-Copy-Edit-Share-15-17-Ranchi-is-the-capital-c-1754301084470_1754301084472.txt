🧠 What is Chat Memory?

Chat memory means the AI remembers previous messages in the conversation — just like how ChatGPT remembers your earlier messages in the same chat.


---

❌ What Happens Without Memory?

If you only send 1 message, like:

{ "role": "user", "content": "Who is the president of India?" }

The AI will answer, but won’t remember this message in the next question.


---

✅ How to Add Chat Memory?

To give memory, you must send all previous messages in the messages array for each new API call.


---

📦 Example:

1️⃣ No Memory (Bad)

messages: [
  { role: "user", content: "Who is the president of India?" }
]

Next time, if you ask:

messages: [
  { role: "user", content: "What is his age?" }
]

It won’t know who "he" is.


---

2️⃣ With Memory (Good)

messages: [
  { role: "system", content: "You are a helpful assistant." },
  { role: "user", content: "Who is the president of India?" },
  { role: "assistant", content: "Droupadi Murmu is the president of India." },
  { role: "user", content: "What is her age?" }
]

Now the AI knows who "her" refers to. ✅


---

💡 How to Store Memory?

You store the conversation in a variable or database and update it every time:

let chatHistory = [];

function addMessage(role, content) {
  chatHistory.push({ role, content });
}
